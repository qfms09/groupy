第一代 Groupy 功能上够用，但视觉上和 Windows 11 格格不入——圆角、Mica 磨砂效果、深色模式，全都跟 Win11 风格抵触，看着有点别扭。

另外还有几个值得说的细节：

- **鼠标悬停预览**：光标放在标签上就能预览那个窗口里的内容，不用点开就知道在哪
- **拖文件到标签**：直接把文件拖到目标软件的标签上，自动复制，不用开着两个窗口来回拖
- **自动分组规则**：设置好规则，相同类型的软件窗口会自动归到一组，减少手动操作
- **标签颜色标记**：给不同标签组涂上不同颜色，一眼就能区分工作区和娱乐区
